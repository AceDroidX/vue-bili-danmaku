## 使用方法

[https://danmu.nana7mi.link/?roomid=21452505](https://danmu.nana7mi.link/?roomid=21452505)

先将网址 ```roomid``` 后的数字改为你的直播间号，通过obs插入浏览器，填写修改后网址，设置宽度 ```400``` ，高度自定，推荐 ```600```
